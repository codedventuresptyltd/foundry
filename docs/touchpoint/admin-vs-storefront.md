# Admin vs Storefront

:::info Coming Soon
Detailed layer documentation coming soon.
:::

Touchpoint provides two interconnected layers built from the same component system.

## Admin Layer

Management interface for:

- Product catalog configuration
- Pricing rules
- Fulfillment zones
- Customer accounts
- Order processing

## Storefront Layer

Customer-facing experience for:

- Product discovery
- Real-time ordering
- Account management
- Order tracking
