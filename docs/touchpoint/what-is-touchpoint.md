# What is Touchpoint?

:::info Coming Soon
Comprehensive Touchpoint overview coming soon.
:::

Touchpoint is a modern UI framework for building B2B commerce experiences.

## Core Capabilities

- Slot-based component architecture
- Real-time pricing integration
- Dynamic search and filtering
- Multi-tenant customization
- Mobile-responsive design

## Target Use Cases

- Trade and distribution
- Manufacturing and industrial equipment
- Food service and hospitality
- Any B2B ordering scenario requiring sophistication
