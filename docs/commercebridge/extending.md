# Extending CommerceBridge

:::info Coming Soon
Extension and customization guides are being prepared.
:::

CommerceBridge is designed to be extended and customized for specific business needs.

## Extension Points

### Custom Workers

Build specialized workers for your business logic.

### Bridge Integrations

Add new shared integrations to the Bridge.

### Custom Pricing Logic

Implement domain-specific pricing rules.

### Fulfillment Strategies

Create custom fulfillment allocation strategies.

## Best Practices

- Follow the architecture patterns
- Keep integrations in the Bridge when multi-tenant
- Use Worker service files for client-specific code
- Document your extensions
